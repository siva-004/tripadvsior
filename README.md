# WanderNest Travel Guide Website

## Project Overview

WanderNest Travel Guide is a modern and responsive travel website created using simple HTML and CSS.  
This project helps users explore beautiful tourist destinations, hotels, restaurants, and travel packages around the world.

The website includes modern UI design, hover effects, travel cards, background images, and responsive layouts for all devices.

---

## Technologies Used

- HTML5
- CSS3

---

## Features

- Responsive Navigation Bar
- Hero Section with Background Image
- Tourist Destination Cards
- Hotel and Restaurant Listings
- Hover Effects and Simple Animations
- Search Bar and Filters
- Travel Package Pricing
- Mobile Friendly Design
- External Links using `target="_blank"`
